Arduino

Danna Paola García Sánchez y Selene Román Celis
Ing Mecatrónica
Fecha: Septiembre 22, 2023
Arduino Electrónica

- Resumen -

Durante esta practica aprendimos acerca de Arduino, su utilidad, cómo funcionan, la manera correcta en la que se debe conectar a los circuitos.

Se realizaron diferentes combinaciones de códigos y armado en los circuitos para que el Arduino realizara diferentes simulaciones.

- Introducción -

Arduino es una plataforma de prototipos electrónicos que facilita el uso con un hardware y software sencillo, Arduino nos ayuda mediante la recepción de una gran variedad de sensores.

En este apartado se verá el proceso que se llevo para armar algunos circuitos que usaran Arduino, para comenzar a familiarizarnos con su utilidad.

- Materiales -

-Arduino uno R3

-Pulsador

-Resistencia

-Led

-Cátodo visualizador

- Desarrollo -

Lo primero que realizamos fue ubicar los componentes necesarios para llevar a cabo nuestra práctica, después debíamos pasar el diseño que teníamos en Tinker Cad a nuestro protoboard para después pasar el código a Arduino. .

Simulación

La práctica se realizó basándonos en circuitos ya armados, por lo que no tuvimos que realizar una simulación debido a que ya teníamos conocimiento de que los circuitos eran funcionales.

Diseño

Contábamos con el diseño de cada circuito a realizar, por lo que nuestra tarea era más sencilla porque teníamos la seguridad de que el diseño funcionaba y solo teníamos que construirlo.

Construcción

Tuvimos que realizar varias construcciones para cada circuito, pero debido a que tenían pocos componentes al momento de construirlo la tarea era más sencilla, gracias a los diseños con los que ya contábamos, solo teníamos que colocar los componentes en la posición que ya tenían para que nuestra construcción funcionará.

- Resultados -

Al terminar logramos realizar 18 salidas diferentes con Arduino y se pudo notar como varios circuitos tenían el mismo armado, pero al modificar el código la funcionalidad del circuito cambiada completamente.

Función circuito 0: Sirve para prender un led y dependiendo del delay que cambies en el código se puede aumentar o disminuir la velocidad.

Función circuito 1: El armado es el mismo que el anterior pero el código se cambia para que el led del arduino se quede encendido .

Función circuito 2: El código se cambia para que el led del arduino no prenda.

Función circuito 3: El led del arduino parpadea más rápido con este código ya que el delay es más corto.

Función circuito 4: Sirve para encender y apagar un led que se conecta directamente a las entradas del arduino.

Función circuito 5: En este circuito se conecta el arduino a un protoboard para colocar un led y de igual manera hacerlo prender y apagar.

Función circuito 6: La función es igual al del circuito 5 con la diferencia de que cambian las fuentes de energía, entradas y salidas entre el protoboard y el arduino.

Función circuito 7: En este circuito se utilizan 2 leds para que enciendan y apaguen los dos al mismo tiempo.

Función circuito 8: El armado es el mismo del circuito anterior pero el código se modifica para que los leds prendan y apagan a destiempo.

Función circuito 9: En el circuito se conecta un display de 7 segmentos cátodo para que al momento de hacerlo funcionar con el código indicado se encienda una línea horizontal en el cátodo.

Función circuito 10: El armado es el mismo que el anterior pero el código cambia para que el display muestre los número 0,1 y 2.

Función circuito 11: En este armado utilizamos un led y un pulsador para que al momento de accionar el pulsador el led prenda y cuando se suelte se apague.

Función circuito 12: En este circuito agregamos un led y un pulsador, las conexiones entre el arduino y el protoboard aumentan ya que utilizamos más elementos y cada pulsador enciende un led diferente.

Función circuito 13: Para este circuito utilizamos el mismo armado que el del circuito 11 y solo cambia el código agregando más componentes pero la función sigue siendo la misma, mientras apretemeos el pulsador prende y si lo soltamos se apaga.

Función circuito 14: En este circuito ocupamos dos leds y dos pulsadores, ambos leds solo prenden cuando los dos pulsadores esten accionados.

Función circuito 15: Ahora utilizamos un led y dos pulsadores, el led prenderá con cualquier pulsador.

Función circuito 16: El circuito solo funcionará cuando esten presionados ambos pulsadores al mismo tiempo para prender el led.

Función circuito 17: Este armado es el más grande, se utilizan 4 leds y un solo pulsador, por cada click irán prendiendo los leds en orden, al oprimir 5 veces después de que todos los leds esten encendidos se apagarán y se podrá repetir el ciclo de nuevo.

- Conclusiones -

Con ayuda de un arduino se pueden realizar circuitos que cambien su funcionalidad dependiendo del programa.

Se pueden hacer circuitos que tengan la misma construccion pero al momento de hacerlos funcionar tengan diferentes acciones, como se pudo observar en la elaboración de la práctica.
