Brazo Robótico

Danna Paola García Sánchez y Selene Román Celis
Ingeniería Mecatrónica
Fecha: Octubre 20, 2023
Arduino Electrónica

- Resumen -

En esta práctica lo que realizamos fue un brazo robótico que funciona a través de 5 servomotores controlados por potenciómetros conectado a un protoboard y una placa arduino.

- Introducción -

Un servomotor es un dispositivo similar a un motor de corriente continua que tiene la capacidad de ubicarse en cualquier posición dentro de su rango de operación, y mantenerse estable en dicha posición.

Lograr coordinar 5 servomotres nos permitirá poder controlar correctamente el brazo robótico y hacer distintas maniobras como levantar objetos y poder pasarlos de un brazo a otro.

- Materiales -

-Servomotor.

-Arduino.

-Protoboard.

-Potenciómetro.

- Desarrollo -

Lo primero que hicimos fue realizar un boceto para visualizar bien como se iba a diseñar el brazo robótico. Al tener un prototipo realizamos por igual cada boceto de todas las piezas que se iban a necesitar para la construcción. Se procedió a realizar todas las piezas en CATIA.

Después se llevaron al corte laser. Proseguimos con el ensamblado de todo para concectar los servomotores. Al ir realizando el ensamble algunas piezas del diseño original se fueron cambiando para mejoría del brazo.

También hubo cambios en la colocación de los servomotores ya que se realizaron pruebas para verificar que cada servomotor cumpliera su función y algunos no funcionaban de la manera deseada.

El circuito y el programa que realizamos para que funcionaran todos los servomotores los diseñamos en base a lo que aprendimos en la práctica 2 de arduino. Por último, concectamos todos los servomotores y realizamos las últimas pruebas para verificar que funcionara correctamente.

Construcción

Para no tener problmas o errores con el brazo robótico se fueron probando uno por uno cada servomotor y verificar su funcionamiento.

Base.- Lo primero que se verificó fue que la base lograra girar corrctamente en 180°.

Brazo 1.- El primer brazo era lo más importante ya que iba a asegurar la gran movilidad de todo el brazo robótico.

Brazo 2.- El segundo brazo nos iba a permitir tener más precisión al momento del alcance a cualquier objeto.

Base garra.- La base de la garra nos ayudaría para poder girar la garra a casi 360° lo que ayuda a ampliar la movilidad del brazo.

Garra .-La garra era uno de los elementos más importantes del brazo ya que si no funcionaba correctamente el agarre o giro podría dificultar su funcionamiento al momento de querer sostener cualquier objeto.

Diseño

Se tuvo que diseñar el circuito completo que íbamos a utilizar con los 5 servomotores, para que cada uno se moviera con un potenciómetro diferente, lo cual se realiza de la siguiente manera, cada uno de los servos tienen 3 cables ( cable negro se conecta a negativo, cable rojo a positivo y cable naranja a un pin digital del Arduino), después se conectan los potenciómetros (terminal 1 a positivo, la de en medio a un pin analógico del Arduino y por ultimo la terminal 2 a negativo), después se conecta el pin de 5V a positivo al protoboard al igual que el pin de GND a negativo del protoboard, después se carga el programa al Arduino y se desconecta la computadora para conectar a la fuente.

Programa

Realizar el código depende de donde colocaste los servos y los potenciómetros en el Arduino.

#include <"Servo.h">

int valor1; //Estas variables sirven para almacenar las lecturas analógicas

int valor2;

int valor3;

int valor4;

int valor5;

int pos1; //Las variables de posición son las que almacenan la posición del servomotor

int pos2;

int pos3;

int pos4;

int pos5;

//Le decimos al codigo que va a existir un servo

Servo myservo1;

Servo myservo2;

Servo myservo3;

Servo myservo4;

Servo myservo5;

void setup() {

Serial.begin(9600);

//Le decimos al codigo donde están conectados los servo

myservo1.attach(3);

myservo2.attach(5);

myservo3.attach(6);

myservo4.attach(9);

myservo5.attach(10);

}

void loop() {

// se lee el valor del potenciómetro

valor1 = analogRead(A0);

valor2 = analogRead(A1);

valor3 = analogRead(A2);

valor4 = analogRead(A3);

valor5 = analogRead(A4);

Serial.println(valor1);

Serial.println(valor2);

Serial.println(valor3);

Serial.println(valor4);

Serial.println(valor5);

//Convertimos el valor del potenciómetro a una posición que entienda el servo

pos1 = map(valor1, 0, 1023, 0, 360);

pos2 = map(valor2, 0, 1023, 0, 360);

pos3 = map(valor3, 0, 1023, 0, 360);

pos4 = map(valor4, 0, 1023, 0, 360);

pos5 = map(valor5, 0, 1023, 0, 360);

//Mandamos la posición a los servos

myservo1.write(pos1);

myservo2.write(pos2);

myservo3.write(pos3);

myservo4.write(pos4);

myservo5.write(pos5);

//esperamos un poco para que se mueva

//Delay nos ayuda a indicar el tiempo en el que nuestro servo va a comenzar a moverse

delay(10); }

- Resultados -

Cuando el brazo robótico estaba completo se realizó la última prueba que consistia en poder alzar una pelota.

También realizamos una prueba con otro brazo para poder pasar la pelota entre los dos brazos. Ambas pruebas funcionaron correctamente.

- Conclusiones -

Realizar esta práctica nos sirvió para aprender más sobre el funcionamiento de Arduino y como utilizarlos de manera más elaborado al tener que controlar distintos servomotores.

También nos ayudó a tener una idea más clara de lo que es el funcionamiento de un brazo robótico ya que juntamos varios conocimientos que hemos ido adquiriendo a través del semestre, como lo es la porgramación, los circuitos, diseño por computadora, la construcción, el corte laser, etc.
