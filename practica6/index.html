- Caja de herramientas -

Materiales

-Madera.

-Clavos.

Desarrollo

Lo primero que se realizo fue un boceto para tener clara la idea que teníamos para planear la manera en la que íbamos a realizarla, después se obtuvieron las medidas que se requerían.

Después con una pluma se marcaron las partes necesarias para la caja y se cortó la madera siguiendo esas líneas, después se lijaron las tablas para que los cortes fueran parejos y al momento de juntar las tablas no quedaran con espacios.

Lo siguiente fue poner los clavos para después cortar un palo que sirve de agarradera.

Conclusión

Realizar la caja de herramientas nos ayudo a perder el miedo de utilizar las maquinas que se encuentran en la explanada del idit.

- Piezas en Solid Works -

Desarrollo

Solid sirve para modelar piezas y ensambles en 3D y planos 2D, es una herramienta muy útil para desarrollar proyectos, Solid cuenta con diferentes herramientas dependiendo de la pieza que quieras modelar.

Para esta pieza se utilizo la herramienta de revolución, para esto se tiene que dibujar el punto de revolución y desde ahí dibujar la mitad de la figura para después utilizar la herramienta siguiendo el eje de revolución.

Para esta pieza primero se hizo la base rectangular y después se volvió a utilizar la herramienta de revolución para hacer la pieza circular de la parte superior, después se utilizo otro croquis en la pieza rectangular para dibujar los círculos y cortarlos.

Conclusión

Solid nos sirve para el desarrollo de piezas, una herramienta muy útil en ingeniería, aunque se necesita dominar cada una de las herramientas que solid ofrece.

- Corte Láser -

Desarrollo

Se realizaron las piezas en solid y en catia, después se pasaron esas piezas a drawing para que se puedan descargar en un archivo DXF, después se llevó a la cortadora laser y ya con las piezas cortadas pensando en la tolerancia necesaria para ensamblar las piezas, se juntaron las piezas para hacer una libélula.

Conclusión

Utilizar la cortadora laser nos ayuda para el desarrollo de proyectos. .

- Impresión 3D -

Desarrollo

La pieza se realizó en SolidWorks con ayuda de varias herramientas como la revolución para el cuerpo y la cabeza, para el brazo se utilizo otra herramienta que se hace con dos planos en los cuales uno se debe utilizar para hacer la circunferencia que se quiere y el otro plano sirve para hacer la guía para que se siguiera el circulo, después a cada piezas se le hicieron unas pestañas para que sirvieran como articulaciones, ya que se tenían todas las piezas se realizó el ensamble y después en cura se puso la pieza para que se pusieran soportes y la pieza se imprimiera correctamente.

- Molde y Contramolde -

Materiales

-Resina.

-Silicón.

-Cera para moldear.

-Catalizador

Desarrollo

Lo primero que se realizó fue una pieza en 3D y después meter la figura en SolidWorks para poner la figura dentro de un recuadro que simula el material, después se metió a VCARVE para que se eligieran los pasos que se querían realizar en el router, después se descargó.

Ya que tuvimos nuestro contramolde en la cera después se hizo una mezcla con silicon y catalizador para que, al colocarla en el contramolde, nos diera un molde el cual se podía usar para sacar figuras de resina .
